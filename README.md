# SignalSim – Cellular Signal Strength Simulator
SignalSim is an interactive web application that realistically simulates cellular network signal strength based on real-world factors like location, network technology, and environment.
Built as a powerful educational and demonstration tool, SignalSim allows users to explore how 5G, 4G, 3G, and 2G signals behave across different terrains. By simply clicking or dragging on an interactive map, users can instantly see how their signal strength, download speed, latency, and connection quality change in real time.
Key Features

Interactive Map: Centered on Lagos, Nigeria with draggable location marker
Multiple Network Types: 5G NR, 4G LTE, 3G UMTS, and 2G GSM
Environment Simulation: Urban, Suburban, Rural, and Highway conditions
Realistic Signal Modeling: Calculates signal strength (dBm) using distance, obstacles, and network capabilities
Live Metrics: Signal bars, estimated speed (Mbps), latency (ms), and connection quality
Visual Feedback: Dynamic signal bars and color-coded status indicators
Cell Tower Visualization: See nearby simulated base stations on the map
